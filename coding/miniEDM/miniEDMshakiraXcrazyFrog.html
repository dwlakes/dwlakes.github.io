Raspberry Pi Mini EDM Show | Shakira X Crazy Frog

This is a mini EDM show that I programmed in Python. I remixed Axel F's "Crazy Frog" and Shakira's "Bzrp Session 53" and also transposed the Shakira portions into F minor to match the key signature of "Crazy Frog."

Key Features

  • LED Light Show
  • Crazy Frog and Shakira Remix
  • Melody and Bass Line

Technologies Used

  • Python
  • Raspberry Pi
  • Multithreading
